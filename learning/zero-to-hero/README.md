# React Zero to Hero - Complete Learning Path

If you read this track from start to end and do the exercises, you will learn React from absolute beginner level to confident application builder.

This guide is written using the Chauffeur.Web project as practical context, but it is designed to teach React generally too.

## How to study this path

- Read chapters in order.
- Write code while reading (don’t just consume passively).
- Complete exercises and compare with provided solutions.
- Keep a personal “learning log” with concepts you struggled with.

## What you will be able to do after this

- Build a complete React app with routing and forms.
- Design maintainable component architecture.
- Integrate with APIs confidently.
- Handle loading, error, and empty states professionally.
- Write basic tests and debug real-world issues.
- Ship and evolve features with clean engineering practices.
