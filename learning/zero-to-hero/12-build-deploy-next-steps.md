# 12 - Build, Deploy, and Career-Level Next Steps

## Objective
Move from local learning to production-minded delivery.

## Build pipeline basics

- `npm run build` produces optimized assets.
- Deploy static frontend to CDN/hosting provider.
- Configure environment variables for API endpoints.

## Integration with C# backend

- Keep frontend API client interface stable.
- Replace mock interception with real HTTP base URL.
- Preserve DTO contracts between TS and C# models.

## Capstone project suggestion

Build “Booking Timeline”:

- Customer timeline of requests.
- Manager grouped reservations by date.
- Add endpoint + UI + tests.

## Career growth checklist

- [ ] Can design component architecture
- [ ] Can wire API data + loading/error states
- [ ] Can test feature logic
- [ ] Can explain trade-offs in code reviews
- [ ] Can ship an end-to-end feature independently

## Final advice

Be consistent. Shipping small, clean improvements weekly will make you a strong React engineer faster than chasing advanced topics too early.
